Description

関数 merge :: Ord a => [a] -> [a] -> [a] は，整列されたリストを二つ取り，一つの整列されたリストにして返す関数です． 以下に使用例を示します．

  > merge [2,5,6] [1,3,4]
  [1,2,3,4,5,6]

関数 merge を再帰を用いて定義してください．ただし，関数 insert や isortなど， 整列されたリストを処理する関数は利用してはいけません．
