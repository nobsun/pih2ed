Description

乗算演算子 * の再帰を参考にして，負でない整数に対する冪乗演算子 ^ を定義 してください．また，その定義を使って 2 ^ 3 を簡約してください．

  • (^) :: Int -> Int -> Int

(^) :: Int -> Int -> Int infixr 8 Source #

べき乗演算子

  m ^ 0 = 1                  -- (1)
  m ^ n = m * (m ^ (n - 1))  -- (2)

    2 ^ 3
  = { (2) }
    2 * (2 ^ 2)
  = { (2) }
    2 * (2 * (2 ^ 1))
  = { (2) }
    2 * (2 * (2 * (2 ^ 0)))
  = { (1) }
    2 * (2 * (2 * 1))
  = { (*) }
    2 * (2 * 2)
  = { (*) }
    2 * 4
  = { (*) }
    8

>>> 2 ^ 3
8
