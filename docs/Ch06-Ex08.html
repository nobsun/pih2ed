Description

関数 merge を使って，マージソートを実行する関数 msort :: Ord a => [a] -> [a] を定義してください． マージソートは，引数のリストを二つに分割し，それぞれを整列した後で再び一つに戻すことで整列を実現します． ただし，空リストと要素が一つのリストはすでに整列されていると考えます．
