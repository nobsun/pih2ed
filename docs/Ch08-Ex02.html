occurs :: Ord a => a -> Tree a -> Bool Source #

出現判定

  occurs x (Leaf y) = x == y
  occurs x (Node l y r) = case compare x y of
    LT -> occurs x l
    EQ -> True
    GT -> occurs x r

元の実装は1つのノードで x と y の比較が 2 回になる場合があるが， 新しい実装では 1 回ですむ．
