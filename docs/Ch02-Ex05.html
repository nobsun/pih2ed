Description

プレリュード関数 init は，空でないリストから最後の要素を取り除きます． たとえば，init [1,2,3,4,5] = [1,2,3,4] です．関数 init の定義を二通り示してください．

Synopsis
  • init' :: [a] -> [a]
  • init'' :: [a] -> [a]

Documentation

init' :: [a] -> [a] Source #

init の一つめの定義

init' (x:[]) = []
init' (x:xs) = x : init' xs

>>> init' [1,2,3,4,5]
[1,2,3,4]

init'' :: [a] -> [a] Source #

init のもう一つの定義

init'' = reverse . tail . reverse

>>> init'' [1,2,3,4,5]
[1,2,3,4]
